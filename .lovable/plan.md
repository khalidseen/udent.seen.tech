

# خطة: مخططات SVG تشريحية مختلفة لكل نوع سن

## المشكلة
حالياً جميع الأسنان تستخدم نفس رسم الطاحن (Molar) في مكون `ToothSurfaceSVG`. لكن علمياً كل نوع سن له شكل مختلف تماماً:

- **القواطع (1,2)**: مستطيلة مع حافة قاطعة (Incisal edge) بدل سطح إطباقي
- **الأنياب (3)**: نقطة حادة واحدة (cusp) مع حافة قاطعة
- **الضواحك (4,5)**: حدبتان (buccal + lingual) مع أخدود مركزي واحد
- **الطواحن (6,7,8)**: 4-5 حدبات مع أخاديد متعددة (الشكل الحالي)

## الخطة

### 1. تعديل `ToothSurfaceSVG.tsx` ليقبل `toothNumber`
- إضافة prop `toothNumber: string` (FDI)
- تحديد نوع السن من الرقم الأخير: `1,2 → incisor`, `3 → canine`, `4,5 → premolar`, `6,7,8 → molar`
- تغيير التسمية: القواطع والأنياب تستخدم "Incisal" بدل "Occlusal"

### 2. إنشاء 4 أشكال SVG تشريحية مختلفة

**القواطع (Incisors)**:
```text
  ┌─────────────┐
  │     B       │  ← وجه شدقي عريض
  ├──┬───────┬──┤
  │M │ Incisal│D │  ← حافة قاطعة رفيعة (بدل Occlusal)
  ├──┴───────┴──┤
  │     L       │  ← وجه لساني مع cingulum
  └─────────────┘
شكل بيضاوي ممدود أفقياً
```

**الأنياب (Canines)**:
```text
شكل ماسي/بيضاوي مدبب مع cusp واحد بارز في المنتصف
حافة قاطعة (Incisal) بدل Occlusal
```

**الضواحك (Premolars)**:
```text
شكل بيضاوي أصغر من الطاحن مع حدبتين فقط
أخدود مركزي واحد يفصل buccal عن lingual
```

**الطواحن (Molars)** — الشكل الحالي محسّن:
```text
شكل كبير مع 4-5 حدبات وأخاديد متقاطعة
```

### 3. تعديل `ToothRecordDialog.tsx`
- تمرير `toothNumber` إلى `ToothSurfaceSVG`
- تغيير label "الإطباقي" إلى "القاطع" للقواطع والأنياب
