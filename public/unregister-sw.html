🔧 إلغاء تسجيل Service Worker

📋 تعليمات الاستخدام:

  1. اضغط على الزر أعلاه لإلغاء تسجيل جميع Service Workers
  2. بعد نجاح العملية، أغلق جميع علامات التبويب الخاصة بالتطبيق
  3. افتح DevTools (F12) > Application > Service Workers
  4. تأكد من عدم وجود service workers نشطة
  5. إذا كانت موجودة، اضغط على Unregister
  6. امسح الـ Cache من Application > Cache Storage > حذف الكل
  7. أعد تشغيل الـ dev server: npm run dev
